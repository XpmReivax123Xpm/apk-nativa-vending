# apk-nativa-vending

Bootstrap inicial de arquitectura Kotlin para una app Android kiosk de vending industrial.

## Objetivo de esta base
- Separar responsabilidades por modulos y capas.
- Mantener backend como fuente de verdad del negocio.
- Tratar serial/hardware y modo kiosk como preocupaciones de primer nivel.
- Documentar plan, bitacora y pendientes desde el dia 1.

## Modulos incluidos
- `app`
- `domain`
- `data`
- `integration-backend`
- `integration-serial`
- `kiosk-device`
